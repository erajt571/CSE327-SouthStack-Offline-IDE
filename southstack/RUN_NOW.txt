📋 QUICK START (3 STEPS):
═════════════════════════════════════════════════════════════════════════════

STEP 1: Enable WebGPU in Chrome (2 minutes)
─────────────────────────────────────────────────────────────────────────────
1. Open Chrome
2. Go to: chrome://flags/#enable-unsafe-webgpu
3. Set to: Enabled
4. Click: Relaunch
5. Chrome will restart - WAIT FOR IT


STEP 2: Start HTTP Server (30 seconds)
─────────────────────────────────────────────────────────────────────────────
Open Terminal and run:

    cd /Users/eloneflax/cse327/southstack
    python3 -m http.server 8000

You should see:
    Serving HTTP on 0.0.0.0 port 8000
    [Press Ctrl+C to quit]

✅ Server is running!


STEP 3: Open in Chrome (30 seconds)
─────────────────────────────────────────────────────────────────────────────
1. Open Chrome
2. Go to: http://localhost:8000/
3. You should see the SouthStack Dashboard!
4. Open Console: F12 or Cmd+Option+I (Mac)


🎯 SEND YOUR FIRST PROMPT:
═════════════════════════════════════════════════════════════════════════════

In Browser Console, type:

    ask("Write a hello world program in Python")

Then PRESS ENTER and WATCH THE MAGIC! ✨

The model will respond in the console:

    def hello_world():
        print("Hello, World!")
    
    hello_world()

🆘 TROUBLESHOOTING:
═════════════════════════════════════════════════════════════════════════════

Problem: "WebGPU is not available"
Solution: Go to chrome://flags/#enable-unsafe-webgpu → Enabled → Relaunch

Problem: Server won't start
Solution: Make sure you're in: /Users/eloneflax/cse327/southstack directory

Problem: "Cannot connect to localhost:8000"
Solution: Check server is running (see Step 2 output)

Problem: Model downloads very slowly
Solution: Normal for 500MB+ file. Wait patiently or check internet speed.

Problem: ask() doesn't work
Solution: Wait for "✅ Model loaded successfully" message
